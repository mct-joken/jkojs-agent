# KOJS-Agent
KOJSのWorkerを管理する部分です.

## KOJSv3からの変更
- コンテナにコードの他にテストケースを渡すように
    - テストケース変更時Dockerイメージの更新が不要になりました
- コンテナに正答ファイルを置かないように
    - コード実行時の安全性が上がりました
- APIでテストケースを受け取るように
    - KOJSv3ではできなかったコードテストが可能になりました
## KOJSv5(JK-OJS)からの変更
- APIを廃止し、バックエンドの情報を定期的に取得するように
  - 並列化のためにキューを別途立てる必要がなくなりました
  - 実行〜結果の返却にかかる時間が短くなりました
